# Walker Goddess Collective | Digital Business Card

The official digital presence and brand showcase for Walker Goddess Collective and all six of its branches. Built as a single-file responsive web application.

## About

Walker Goddess Collective is a disability advocacy and consulting firm and the parent company of six branches spanning fashion, wellness, education, publishing, and technology. This web app serves as a digital business card and brand introduction for the entire organization.

## What's Inside

- **Home** -- Walker Goddess Collective overview and branch navigation
- **Advocacy and Consulting** -- Full services menu with pricing and disclaimer
- **Breaking Barriers Co** -- Adaptive fashion and accessibility product showcase
- **Simply U** -- Custom adaptive wigs and organic wellness products
- **Sojourner Homeschool Academy** -- Free K12 homeschool platform overview
- **Denver Legends** -- Publishing imprint and works showcase
- **WGC Studio** -- AI applications and tech products
- **Contact and Inquiry** -- Contact form with direct business contact information

## Tech Stack

- Vanilla HTML, CSS, and JavaScript
- No frameworks, no backend, no dependencies
- Single file deployment
- Netlify Forms for contact form submissions
- Fully responsive, mobile first design with swipe navigation

## Deployment

1. Rename file to index.html
2. Push to GitHub repository
3. Connect repository to Netlify
4. Deploy -- no build command or publish directory needed
